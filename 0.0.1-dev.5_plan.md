## Changed Log
<!--0.0.1-dev.5-tensor-->
_16 - july - 2026_
- The process of developing a tensor consists of three main parts: tensor (contiguous and view), storage, and operations.

- Development of storage (ArrayStorage), which will store arrays (the storage property) and empty/reusable indexes (the empty_idx property).

- ArrayStorage now has the following methods:
  - ArrayStorage::new
  - ArrayStorage::get_storage
  - ArrayStorage::get_mut_storage
  - ArrayStorage::get_empty_idx
  - ArrayStorage::get_mut_empty_idx
  - ArrayStorage::push
  - ArrayStorage::remove

- Development of a contiguous tensor (Tensor), which will store array indexes (the array itself will be stored in ArrayStorage), gradient indexes (the gradient itself will still be an array), and the Reference Count (RC) of ArrayStorage itself.

- Tensor now has one method. new
  - Tensor::new

- Combine the error files into one main error file (PzeudoErr) in pzeudo/error.rs.

- Rename the tensor to pzudo_tensor.

- Added a new initialization method for Arrays, namely Array::zeros

- menambahkan metode inisialisasi baru pada tensor yaitu Tensor::from_vector_with_shape

- implemented Display on Array and ArrayView

- added a new method to the array view

- Separate Metadata into a separate file, namely at /pzeudo_num/metadata.rs

- added ArrayRef, working as a bridge between Tensor and Array operations. has a structure similar to ArrayView but the metadata itself is not an own type but a reference, this is useful for creating a preview data type which aims to connect Tensors which will have 2 modes (contiguous and view)

- For ArrayRef, it has a generic type in the form of T, which can be filled with Contiguos and Views. dalam implementasi metode pada pzeudo_num/_ops, generic type ini lah yang akan membdakan implementasinya jika ingin ada optimalisasi.

- added a new tensor method for initialization namely Tensor::from_vector_with_shape

- ArrayStorage development

- Proses pembauatan Alur Forward, menggunakan RecordLabel untuk merekam setiap operasi yang dilakukan.

- the process of creating arithmetic methods on tensors
